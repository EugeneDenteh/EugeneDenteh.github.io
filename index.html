A Few Accomplishments

These are the projects that I have been able to successfully accomplish using python.

DSPS23 Student competition

My team and I enrolled in the second track of the competition where we were supposed use a set of image data to train a YoloV5 model to detect pavement cracks. We were awarded the second place with a model accuracy of about 0.73.

Password security system

The development of an object-oriented password security system using python. The program is capable of doing the following;

  • It allows users to have only three attempts.
  • It alerts users on the number of attempts left after any failed attempt.
  • It blocks the user after the third attempt.
  • It also prompts the user if the password is not up to the required length and if there are spaces in the password.
